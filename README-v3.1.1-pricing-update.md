# Walnex Creative v3.1.1 Pricing Page Update

This is a page-only update.

## How to deploy

Replace the existing `pricing.html` in your deployed Walnex Creative site with this updated `pricing.html`.

No other files are required for this update because the page-specific v3.1.1 pricing styles are embedded directly in the file.

## Pricing changes included

- Brand Foundation is now $500 one-time with no required monthly support.
- Website Launch is $750 plus $50/month support for 3 months.
- Digital Launch Setup is $750 one-time.
- Added bundle options:
  - Brand & Website Launch
  - Website & Launch Support
  - Identity & Profile Launch
- Complete Business Launch is now $2,000 with 3 months of support included.
- Continued hands-on support after the included period is optional at $100/month.
- Web & Domain Support is a separate $25/month minimum option.
- Removed reference to GitHub Pages from the Web & Domain Support description.
- Added pricing philosophy/core values copy around partnership, care, low upfront cost, and startup support.
